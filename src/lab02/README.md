# Лабораторная работа №2: Коллекция объектов

**Коллекция (контейнер)** — класс, предназначенный для хранения, управления и навигации по группе однотипных объектов. Поддерживает добавление, удаление, поиск, сортировку, итерацию и индексацию.

## Вариант 5: Транспорт
**Сущности:** Car, Bus, Ticket, Route, Driver

**Реализовано:** класс BusFleet (автопарк) для хранения объектов Bus из ЛР-1

## Класс Bus (из ЛР-1):
**Закрытые поля:**

- `_route_number` — номер маршрута (целое число, >0)
- `_capacity` — вместимость автобуса (1-50)
- `_passenger_count` — текущее количество пассажиров (0-50)
- `_current_speed` — текущая скорость (0-120)
- `_state — состояние` (in_depot / on_route / maintenance)

## Бизнес-методы:

- `start_route()` — выезд на маршрут
- `return_to_depot()` — возврат в депо
- `send_to_maintenance()` — отправка на ТО
- `repair_complete()` — завершение ТО
- `board_passengers(count)` — посадка пассажиров
- `alight_passengers(count)` — высадка пассажиров
- `free_seats()` — свободные места
- `load_factor()` — загрузка в процентах

## Класс BusFleet (коллекция):
**Внутреннее хранилище:** self._buses = [] — список объектов Bus

**Методы управления:**

- `add(bus)` — добавить автобус (проверка типа и уникальности номера)
- `remove(bus)` — удалить автобус по объекту
- `remove_at(index)` — удалить автобус по индексу
- `get_all()` — вернуть копию списка всех автобусов

**Методы поиска:**

- `find_by_route_number(route_number)` — поиск по номеру маршрута
- `find_by_state(state)` — поиск по состоянию
- `find_by_load_factor_range(min_factor, max_factor)` — поиск по диапазону загрузки

**Методы сортировки:**

- `sort_by_route_number()` — сортировка по номеру маршрута
- `sort_by_load_factor()` — сортировка по загрузке (убывание)
- `sort_by_passenger_count()` — сортировка по количеству пассажиров (убывание)

**Логические операции** (возвращают новые коллекции):

- `get_on_route()` — автобусы на маршруте
- `get_in_depot()` — автобусы в депо
- `get_available_for_route()` — готовые к выходу (в депо + есть места)

**Магические методы:**

- `__len__()` — длина коллекции (len(fleet))
- `__getitem__(index)` — доступ по индексу и срезам (fleet[0], fleet[1:3])
- `__iter__()` — итерация (for bus in fleet)
- `__contains__(bus)` — проверка наличия (bus in fleet)
- `__str__()` — строковое представление коллекции
