# Лабораторная работа №7: Консольное приложение

## 1. Цель работы

Объединение всех знаний, полученных в ЛР1–ЛР6, в единое работающее приложение. Реализация интерактивного CLI-интерфейса с меню, обработкой пользовательского ввода, собственными исключениями и сохранением данных в JSON.

## 2. Структура проекта

```
src/lab07/
├── main.py          # Точка входа, запуск CLI
├── cli.py           # Интерфейс: меню, ввод/вывод, форматирование
├── app.py           # Бизнес-логика: управление коллекцией
├── storage.py       # Сохранение/загрузка в JSON
└── exceptions.py    # Собственные исключения
```

**Назначение файлов:**

| Файл | Ответственность |
|------|-----------------|
| `main.py` | Запуск приложения |
| `cli.py` | Только ввод и вывод, никакой бизнес-логики |
| `app.py` | Операции над коллекцией, вызов стратегий |
| `storage.py` | Чтение и запись JSON-файла |
| `exceptions.py` | `DuplicateItemError`, `ItemNotFoundError`, `InvalidInputError` |

Приложение использует классы `Bus`, `TouristBus`, `SchoolBus` из ЛР2-ЛР3, коллекцию `TypedCollection` из ЛР6 и стратегии сортировки из ЛР5.

## 3. Описание CLI

### Пункты меню

```
==================================================
АВТОПАРК - Управление автобусами
==================================================
1. Показать все автобусы
2. Добавить автобус
3. Удалить автобус
4. Найти автобус по маршруту
5. Фильтрация автобусов
6. Сортировка автобусов
7. Отправить на маршрут
8. Посадить пассажиров
0. Выход
--------------------------------------------------
```

### Обработка ошибок

- Ввод буквы вместо числа → перехват `ValueError` → сообщение «Ошибка: введите число»
- Добавление существующего маршрута → `DuplicateItemError` → сообщение об ошибке
- Удаление несуществующего маршрута → `ItemNotFoundError` → сообщение об ошибке
- Неверные данные (вместимость >50, скорость >120) → `InvalidInputError`

### Сохранение и загрузка

- При запуске: автоматическая загрузка из `buses.json`
- При выходе (пункт 0): автоматическое сохранение
- Формат: JSON с полями `type`, `route_number`, `capacity`, `passenger_count`, `current_speed`, `state`
